# V18.0 — Blue Current Connect

Built on the uploaded V17.0 baseline.

## Added
- Connector SDK and registry
- Eight connector adapters and health states
- Connector Hub with latency, uptime and event counters
- Live Event Inspector
- API Playground that publishes into Event Bus and App State
- Integration Marketplace
- AI Integration Monitor
- Security Center
- Connector health pulse presentation

## Architecture
All external events enter through the Connector SDK, are normalized, emitted through the Event Bus and may update shared App State. No connector directly updates another product module.
