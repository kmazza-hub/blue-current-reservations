# Blue Current V23.0 — Authentication & Organizations

## Implemented

- Password hashing with Node.js scrypt
- Hashed bearer-session records with expiration
- Login, logout, session restore, and active-user endpoints
- Protected API routes
- Server-enforced role permissions
- Tenant-scoped bootstrap, reservations, audit logs, configurations, and invitations
- Organization-aware sessions and organization switching
- Location-level access restrictions
- Team invitation creation
- Login overlay and access-administration dashboard
- Demo owner, general-manager, and second-tenant accounts

add HTTPS, secure cookies, CSRF protection, email delivery, rate limiting,
managed secrets, password reset, MFA, and a managed identity/database platform.
