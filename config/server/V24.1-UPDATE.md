# Blue Current V24.1 — Waitlist & Reservations

## Implemented

- Operational reservation list with status filters
- Reservation detail editor
- Confirmed, arrived, seated, completed, cancelled, and no-show states
- VIP flagging
- Accessibility notes
- Guest notes and source tracking
- Compatible-table matching
- One-click reservation seating
- Persistent reservation changes
- Reservation event history and audit records
- Live reservation updates over server-sent events
- New reservation creation from the host stand
- App State synchronization for covers and arrival metrics
- Tight integration with V24.0 Live Floor Operations

## API

- `GET /api/reservation-operations?locationId=...`
- `POST /api/reservation-operations`
- `PATCH /api/reservation-operations/:id`
- `POST /api/reservation-operations/seat`
