# Blue Current V15.1 — Mission Control

## Added
- Live Mission Control section driven by the Event Bus
- Real-time operational event feed
- Shared-state status panel
- Event, occupancy, reservation, call, and recovered-revenue metrics
- Replay and clear-feed controls
- Responsive desktop, tablet, and mobile layouts

## Architecture
Mission Control is a read-only projection of Event Bus activity and App State. It does not own reservation data or directly update other modules.
