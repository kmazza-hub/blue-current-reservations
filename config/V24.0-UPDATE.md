# Blue Current V24.0 — Live Floor Operations

## Implemented

- Persistent restaurant floor tables
- Drag-and-drop table positioning
- Table states: available, reserved, seated, cleaning, blocked
- Table inspector with guest, party, section, and server controls
- Waitlist creation
- Automatic compatible-table seating from the waitlist
- Occupancy, active-table, availability, and queue metrics
- Persistent seating-event history
- Server-enforced location permissions
- Audit events for floor changes
- Server-sent real-time floor events
- App State and Event Bus synchronization
- Responsive desktop and tablet floor interface
- Correct distributed Sign in and Sign out button classes

## API

- `GET /api/floor?locationId=...`
- `PATCH /api/floor/tables/:id`
- `POST /api/floor/waitlist`
- `POST /api/floor/seat-waitlist`
