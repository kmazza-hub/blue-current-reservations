# Blue Current Restaurant Website

A clean, responsive static website for Blue Current's AI restaurant reservation platform.

## Files

- `index.html` — page content and structure
- `styles.css` — all styling and responsive layouts
- `script.js` — mobile menu and automatic footer year

## Run locally in VS Code

1. Open this folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Choose **Open with Live Server**.

You can also open `index.html` directly in a browser, but Live Server makes editing easier.

## Git setup

Run these commands from the project folder:

```bash
git init
git add .
git commit -m "Build new Blue Current restaurant website"
```

To connect a new GitHub repository:

```bash
git branch -M main
git remote add origin YOUR_GITHUB_REPOSITORY_URL
git push -u origin main
```

## Before launch

Update the email address in the final CTA inside `index.html` if needed:

```html
mailto:hello@bluecurrentco.com
```
