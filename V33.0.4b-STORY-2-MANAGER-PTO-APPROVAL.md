# Blue Current V33.0.4b — Story 2

## Manager PTO Approval

### User story
As a manager, I can review an employee's pending time-off request and approve or deny it with an optional comment.

### Included
- Employee Portal PTO requests now appear in the manager Workforce PTO Queue.
- Pending requests show employee name, request type, dates, submitted date, and employee note.
- Managers can approve or deny a pending request.
- Managers can include an optional comment of up to 300 characters.
- The server rejects repeat decisions on requests that are no longer pending.
- Employees see the updated status and manager comment after refreshing their portal.
- Pending requests are sorted ahead of completed requests.

### Test checklist
1. Sign in to the Employee Portal and submit a new time-off request.
2. Sign in to the main Blue Current application as a manager.
3. Open **Workforce** and locate **PTO Queue**.
4. Confirm the employee request appears with the correct name, type, dates, and note.
5. Add an optional manager comment and choose **Approve** or **Deny**.
6. Confirm the request leaves the pending state and the Pending PTO KPI decreases.
7. Refresh the Employee Portal and confirm the new status and manager comment are visible.
