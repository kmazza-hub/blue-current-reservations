# Blue Current V33.0.1 — Workforce Foundation

## Customer value
Managers can create employee profiles, record weekly availability, submit and decide PTO requests, and create reusable shift templates.

## Added
- Workforce Foundation service and authenticated APIs
- Employee profile creation and updates
- Availability upsert by employee/day
- PTO request and approval workflow
- Shift-template creation
- Manager-facing Workforce Foundation UI
- Realtime workforce events and audit records

## API
- `GET /api/workforce-foundation`
- `POST /api/workforce-foundation/employees`
- `PATCH /api/workforce-foundation/employees/:id`
- `POST /api/workforce-foundation/availability`
- `POST /api/workforce-foundation/pto`
- `PATCH /api/workforce-foundation/pto/:id`
- `POST /api/workforce-foundation/shift-templates`

## Next sprint
V33.0.2 will consume these records in the weekly scheduling engine.
