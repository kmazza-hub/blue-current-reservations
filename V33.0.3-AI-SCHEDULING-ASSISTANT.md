# Blue Current V33.0.3 — AI Scheduling Assistant

## Customer value
Managers now receive explainable staffing recommendations directly above the weekly schedule. Blue Current ranks qualified employees for open shifts, detects projected overtime, evaluates PTO, availability, overlaps, role fit, and compares dinner staffing with reservation demand.

## Included
- Smart Fill candidate ranking
- One-click application of the best candidate
- Overtime-reduction recommendations
- Reservation-demand coverage recommendations
- Explainable reason codes and projected weekly hours
- Realtime refresh after an AI recommendation is applied
- Authenticated Smart Fill API

## Guardrails
The assistant does not silently modify a schedule. Every recommendation is visible and requires manager confirmation before application.
