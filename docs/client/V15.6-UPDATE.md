# Blue Current V15.6 — Time Machine

Adds a state-driven service replay system that can rewind and replay a full dinner service across the shared App State architecture.

## Included
- Interactive service scrubber from 6:00 PM through 8:24 PM
- Nine synchronized operational snapshots
- Play, pause, jump-to-moment, and Return Live controls
- Live occupancy, reservation, and revenue pace readouts
- Digital Twin table-state replay for Table 14
- Executive and shared UI synchronization through App State
- Event Bus telemetry for preview and live-restoration events
