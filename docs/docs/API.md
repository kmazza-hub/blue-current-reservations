# Blue Current Cloud V22 API

- `GET /api/health`
- `GET /api/bootstrap`
- `GET /api/reservations`
- `POST /api/reservations`
- `GET /api/audit`
- `POST /api/audit`
- `PATCH /api/configurations/:id`
- `GET /api/events` — server-sent event stream

The V22 server uses only Node.js built-in modules and requires no package installation.
