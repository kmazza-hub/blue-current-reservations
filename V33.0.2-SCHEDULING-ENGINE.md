# Blue Current V33.0.2 — Scheduling Engine

## Customer value
Managers can create assigned or open shifts, navigate weeks, copy the previous week, validate availability/PTO/overlap/overtime, review hours and projected labor, and publish the schedule.

## API
- `GET /api/scheduling`
- `POST /api/scheduling/shifts`
- `PATCH /api/scheduling/shifts/:id`
- `DELETE /api/scheduling/shifts/:id`
- `POST /api/scheduling/copy-previous`
- `POST /api/scheduling/publish`

## Notes
Existing V33.0.1 workforce records are reused. New collections (`scheduleShifts`, `schedulePublications`) are created lazily in the file-backed database.
