# Blue Current V33.0.4b — Story 1: Employee PTO Request

## Purpose
Complete one focused employee story: an employee can submit a structured time-off request from a phone and immediately see it as pending.

## Changed files
- `client/employee.html`
- `client/js/employeePortal.js`
- `server/services/employeePortalService.js`
- `package.json`

## Improvements
- Labeled start and end date fields with mobile-friendly date controls.
- Vacation, sick, personal, and unpaid request types.
- Optional manager note with a 300-character limit.
- Client-side date validation and inline status messaging.
- Disabled/loading submit state to reduce duplicate submissions.
- Server-side date, type, note-length, and duplicate-request validation.
- Request type displayed in PTO history.
- Successful requests appear immediately with `pending` status.
