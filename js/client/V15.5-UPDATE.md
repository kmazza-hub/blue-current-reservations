# Blue Current V15.5 — Guest Journey Live

Adds a state- and event-driven guest lifecycle visualization spanning call, recognition, inventory matching, confirmation, arrival, seating, dining, and follow-up.

## Included
- Eight-stage live guest journey
- Full-journey presentation mode
- Journey history feed
- Digital Twin table transitions during arrival, seating, and dining
- Mission Control support for post-reservation lifecycle events
- Shared Event Bus integration
