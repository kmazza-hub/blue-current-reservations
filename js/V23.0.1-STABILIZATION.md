# Blue Current V23.0.1 — Stabilization

## Corrected

- V23 authentication CSS now uses a new cache version.
- V23 Cloud API now uses a new cache version.
- Local server sends `no-store` for HTML, CSS, and JavaScript.
- Authentication validates its required Cloud API methods before login.
