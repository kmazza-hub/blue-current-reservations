# Blue Current V24.2 — Staff & Sections

## Implemented

- Staff roster with active, break, and off states
- Section assignments
- Server workload tracking
- Active-table and cover counts
- Capacity-based workload utilization
- Overload detection
- Persistent section changes
- Persistent server assignments on floor tables
- Staff detail editor
- Shift and role controls
- Server reassignment events
- Real-time staff and section events
- App State synchronization
- Tight integration with Live Floor Operations

## API

- `GET /api/staff-operations?locationId=...`
- `PATCH /api/staff-operations/staff/:id`
- `POST /api/staff-operations/assign-section`
- `POST /api/staff-operations/reassign-table`
